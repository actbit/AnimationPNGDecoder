# AnimationPNGDecoder

Animated Portable Network Graphicsをデコード及びアニメーションを実行するためのツールです。

## 操作方法

起動後テキストボックスに`.png`もしくは`.apng`ファイルのパスを入力します。
**画像として読み込み**ボタンを押すことで通常のpngファイルとして読み込みます。
**アニメーションとして読み込み**ボタンを押すことでapngファイルとして読み込みます。

## その他への実装

Decoderフォルダ内にある実装は.net standard 2.0に批准したコードと同等であるため他のプラットフォーム向けに再実装可能です。
libraryとしての提供も検討中です。
